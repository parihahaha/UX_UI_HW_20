Native ARC Case Study

Responsive Website Redesign

Projects overview

The Native Animal Rehabilitation Centre (Native ARC) website fails to communicate a sense of trust and professionalism to visitors due to inconsistent texts, layouts, UI components and broken hyperlinks. Responsive website redesign will help Native ARC communicates its values and professionalism to the users so the website can recruit more volunteers and receive more donation.
